Credit Card Fraud Detection

A Machine Learning Project by Khushi Katwe

Project Overview

This project detects fraudulent credit card transactions using Machine Learning techniques. It includes data preprocessing, EDA, model training, ROC curve analysis, and SHAP explainability.

Key Features

  • Data preprocessing & feature scaling
  • Class imbalance handling
  • Model training & evaluation
  • Confusion Matrix visualization
  • ROC Curve & AUC Score
  • SHAP Explainability Plot
